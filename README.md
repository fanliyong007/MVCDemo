# MVCDemo

1、采用MVC模式开发的网站（第一个网站项目）
======================================
## 功能：会员增、删、改、查

* 查询所有的会员信息
* 根据name、address、phone可以精确查询
* 能够进行模糊查询
* 能够进行会员添加
* 能够进行会员删除，删除之前要给出提示
* 能够进行会员修改，对会员名进行唯一判断

2、使用到的技术
==============

* MVC设计模式：JSP Servlet POJO
* 数据库使用Mysql
* 连接数据库、JDBC、C3P0数据库连接池
* JDBC工具采用DBUtils
* 提示操作使用JQuery

3、难点：
=======

* （1）多个请求如何使用一个Servlet
* （2）如何进行模糊查询
* （3）如何在创建或修改的时候，验证用户名是否已被占用，并给出提示
